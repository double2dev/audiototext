# Contributing to FreeAudioToText

Thank you for your interest in helping improve FreeAudioToText! Even though our core processing engine is closed-source to protect our proprietary transcription architecture, this public repository is essential for improving the user experience.

## How Can I Contribute?

### 1. Reporting Bugs
If you find a bug in the web interface or transcription results, please open an issue using the **Bug Report** template. Provide as much detail as possible, including:
- Steps to reproduce
- Expected behavior vs. actual behavior
- Browser and OS version

### 2. Suggesting Enhancements
Have an idea for a new feature? We'd love to hear it! Use the **Feature Request** template to explain your idea, why it would be useful, and how it improves the FreeAudioToText experience.

### 3. Helping Others
You can contribute greatly by answering questions from other users in the issue tracker or confirming if you've experienced similar bugs.

## Issue Etiquette
- Please search existing issues before opening a new one to avoid duplicates.
- Be clear and descriptive in your issue titles.
- Treat everyone with respect. See our [Code of Conduct](CODE_OF_CONDUCT.md).

Thank you for helping make audio transcription fast, free, and accessible!
